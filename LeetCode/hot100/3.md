## [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description)

> Tag: 双指针

### 方法一：双指针
* 时间复杂度: ${O(n)}$
* 空间复杂度: ${O(1)}$
```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<int, int> cnt;
        int res = 0;
        // two pointers: [j, i]
        for (int i = 0, j = 0; i < s.size(); i++) {
            cnt[s[i]]++;
            while (cnt[s[i]] > 1) {
                cnt[s[j]]--;
                j++;
            }
            res = max(res, i - j + 1);
        }
        return res;
    }
};
```
