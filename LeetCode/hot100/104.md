## [104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/description)

> Tag: 递归

### 方法一：递归
* 时间复杂度: ${O()}$
* 空间复杂度: ${O()}$
```cpp
class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (!root) return 0;
        return 1 + max(maxDepth(root->left), maxDepth(root->right));
    }
};
```
