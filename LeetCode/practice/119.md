## [119. 杨辉三角 II](https://leetcode.cn/problems/pascals-triangle-ii/description/)

> Tag: 模拟

### 方法一：模拟
* 时间复杂度: ${O()}$
* 空间复杂度: ${O()}$
```cpp
class Solution {
public:
    vector<int> getRow(int rowIndex) {
        vector<int> ans = {1};
        for (int i = 1; i <= rowIndex; i++) {
            vector<int> row = {1};
            for (int k = 0; k < ans.size() - 1; k++) {
                row.push_back(ans[k] + ans[k + 1]);
            }
            row.push_back(1);
            ans = row;
        }
        return ans;
    }
};
```
