## [1047. 删除字符串中的所有相邻重复项](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/description/)

> Tag: 栈 模拟

### 方法一：栈

* 时间复杂度: ${O(n)}$
* 空间复杂度: ${O(n)}$

string 有 push_back(c) 和 pop_back() 

```cpp
class Solution {
public:
    string removeDuplicates(string s) {
        string stk;
        for (char c: s) {
            int n = stk.size();
            if (n >= 1 && c == stk[n - 1]) {
                stk.pop_back();
            } else {
                stk.push_back(c);
            }
        }
        return stk;
    }
};
```
